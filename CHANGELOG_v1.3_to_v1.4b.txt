================================================================================
                              RÉSUMÉ EXÉCUTIF
================================================================================

La version 1.4b apporte un nouveau modèle de transcription ultra-rapide,
une fenêtre Paramètres entièrement redessinée et une interface française
corrigée avec tous les accents. Plus de 800 passes de vérification ont été
effectuées pour garantir la stabilité.

Gains principaux :
  • Nouveau modèle large-v3-turbo (2× plus rapide)
  • Fenêtre Paramètres redessinée (style cohérent)
  • Interface française avec accents complets
  • Sauvegarde config atomique renforcée

================================================================================
                         NOUVELLES FONCTIONNALITÉS
================================================================================

1. MODÈLE LARGE-V3-TURBO
   ──────────────────────
   • 2× plus rapide que large-v3
   • Qualité de transcription quasi identique
   • Disponible directement dans les Paramètres

2. FENÊTRE PARAMÈTRES REDESSINÉE
   ──────────────────────────────
   • Style cohérent avec la fenêtre principale (sans barre Windows)
   • Barre de titre custom avec bouton ✕ et drag
   • Coins arrondis, transparence identique

3. INTERFACE FRANÇAISE CORRIGÉE
   ────────────────────────────
   • Tous les accents restaurés (Paramètres, Modèle, Périphérique, Prêt...)
   • Traductions complètes FR / EN / DE

================================================================================
                         BUGS CORRIGÉS
================================================================================

1. FIX ensure_ascii=False
   ───────────────────────
   Problème : Les accents étaient échappés dans le fichier config JSON
   Solution : ensure_ascii=False dans json.dump()
   Impact   : Accents correctement sauvegardés dans la config

2. FIX ButtonRelease-1
   ────────────────────
   Problème : Curseur de resize non réinitialisé après relâchement
   Solution : Bind ButtonRelease-1 pour reset du curseur
   Impact   : Curseur revient à la normale après un resize
