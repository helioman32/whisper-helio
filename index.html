Support

Dépannage

Les problèmes les plus courants et leurs solutions.

Aucun microphone détecté
  • Vérifier que le micro est branché et sélectionné dans Windows
  • Redémarrer Whisper Hélio
F9 ne répond pas
  • Lancer WhisperHelio.exe en tant qu'Administrateur
  • Changer le raccourci dans les Paramètres (F10, F11, F12)
Transcription lente
  • Passer au modèle large-v3-turbo dans les Paramètres
  • Vérifier que CUDA est bien détecté (Périphérique → auto)
Antivirus bloque le .exe
  • Faux positif dû à la compilation Nuitka — code source vérifiable sur GitHub
  • Ajouter WhisperHelio.exe en exclusion antivirus
Fenêtre invisible au démarrage
  • Supprimer %USERPROFILE%\whisper_helio_config.json
  • Relancer l'application — la fenêtre revient en position par défaut
